# frontend-design：UI 审美加速器

> 内置 Shadcn/Tailwind 等主流规范，生成工业级前端代码

## 核心功能

frontend-design 是 UI 审美加速器，内置 Shadcn/Tailwind 等主流规范，生成具备高级交互感的工业级前端代码。

## 解决什么问题

想给项目做一个 Dashboard 或落地页，但生成的界面丑到没法直接用？

## 能力提升

- 内置主流 UI 规范（Shadcn/Tailwind）
- 生成的代码具备高级交互感
- 工业级质量，不是 demo 级别

## 适用场景

- 需要快速生成美观的 UI 组件
- 想要遵循现代前端最佳实践
- 需要在项目中保持 UI 一致性
