# agent-browser：让 AI 接管浏览器

> 赋予 AI 浏览器控制权，自动点击、模拟登录、定向抓取结构化数据

## 核心功能

agent-browser 是 Claude Code 生态中最值得优先装的 skill 之一。它赋予 AI 浏览器控制能力，让 Claude Code 不再局限于代码编辑器，而是能够真正"执行"任务。

## 解决什么问题

- 登录后台、点按钮、填表单
- 截图、抓取页面信息
- 测试 Web App
- 跑自动化流程

## 核心价值

它的价值不在"会打开网页"，而在于它把网页操作变成了 Claude Code 可以连续执行的一条工作流。

- 做测试的人会用到
- 做调研的人会用到
- 做运营、抓取、后台操作的人也会用到

```bash
npx skills add vercel-labs/agent-browser@agent-browser -g -y
```
