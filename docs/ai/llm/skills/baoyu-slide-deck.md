# baoyu-slide-deck：17种PPT风格，一个 skill 搞定

> 来源：微信公众号「大瑜」

## 背景

AI 生成 PPT 这两年基本上也算是突飞猛进的发展。特别是 Gemini 的 banana 出来之后，大家生成 PPT 的风格一下子多样子起来了。

以前很多人做出来的页面都差不多，模板味很重。

但是，现在你想输出这六种 PPT 风格也可以，输出那六种 PPT 风格也可以。

然而，你想写一个好的 PPT，需要自己去补充提示词，提示词的质量和 PPT 的精美度直接挂钩。很多人都卡在这一步！

今天大瑜分享一个很好用的 skill 完美解决这个问题：**baoyu-slide-deck**。

## skill 介绍

这个 skill 最方便的是：你只需要把 PPT 大纲上传上去，后面它会一步步引导你完成。

你不用去找：
- 商务感的 ppt
- 杂志感的 PPT

这个 skill 会一步一步引导你完成，你只要选择你喜欢的风格，剩下的都交给 skill 去跑吧。

一共 **17 种 PPT 风格**，囊括了市场上大部分的风格。你只需要按照提示内容输入你的想法即可。

## 使用方法

### 第一步：找到 skill 的文件，在你的 Claude Code 对话框输入下面的内容：

```
帮我安装这个 skill：
npx skills add https://github.com/jimliu/baoyu-skills --skill baoyu-slide-deck
```

### 第二步：等待安装好之后，直接将你的大纲 md 文件丢上去

### 第三步：按照提示，选择风格，一会，这个 PPT 就做好了
