# skill-creator：工作流沉淀利器

> 真正高阶的用法，是把自己的工作流做成 skill

## 核心功能

所有会长期用 Claude Code 的人，最后都会走到这一步：不再满足于只装别人做好的东西，而开始把自己的工作方式沉淀成 skill。

## 解决什么问题

今天你可能反复在做这些动作：

- 写提纲
- 改标题
- 统一格式
- 补测试
- 生成 release notes
- 拆分任务步骤

如果某个动作一周会重复很多次，那它就值得被固化。

## 核心价值

skill-creator 做的，就是把这些重复动作从"零散经验"变成"可复用流程"。

当 Claude Code 开始复用你的工作流，它就不再只是一个会听命令的助手，而更像一个逐渐熟悉你做事方式的同事。你沉淀得越多，它的产出越稳定。

对重度用户来说，skill-creator 不只是一个工具，而是把经验转化成资产的开始。

## 安装

```bash
npx skills add anthropics/skills@skill-creator -g -y
```
