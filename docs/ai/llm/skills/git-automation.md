# git-automation：工程化 Commit

> 自动提取 Diff 信息，生成符合工业标准的 Commit 消息与 PR 说明文档

## 核心功能

git-automation 能自动提取 Diff 信息，生成符合工业标准的 Commit 消息与 PR 说明文档。

## 解决什么问题

- commit message 不知道怎么写才规范
- PR 说明文档总是敷衍了事
- 每次都要花时间想怎么描述改动

## 能力提升

- 自动分析代码变动
- 生成符合团队规范的 commit message
- 自动生成清晰的 PR 描述
