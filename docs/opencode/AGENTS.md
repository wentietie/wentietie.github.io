# docs/opencode

## Purpose

OpenCode is an open-source AI coding assistant, often referred to as the "open-source version of Claude Code". It provides terminal-based AI programming assistance with support for 75+ models and has gained significant popularity with 6.4万 GitHub Stars and 100万+ monthly active developers.

## Key Concepts

- **OpenCode**: Open-source alternative to Claude Code, designed for terminal-based coding
- **Oh-My-OpenCode**: Important plugin/enhancement for OpenCode that enables multi-agent workflows
- **Multi-model Support**: Supports 75+ different AI models
- **Zed Integration**: Can be used with Zed IDE for graphical code editing via Agent Servers

## Use Cases

- Complex project development from scratch (0到1)
- Large-scale coding tasks
- Multi-agent collaboration workflows
- Terminal-based AI programming
